# Cost to Serve Calculator

A service design framework for calculating the true cost of delivering services.

Created by **Gerry Scullion** of [This is HCD](https://www.thisishcd.com)

## About

Cost to Serve (CTS) is a framework that helps organisations understand and visualise the real costs involved in delivering services. It bridges service design thinking with business language by quantifying:

- **Direct costs** - Actor/role salaries broken down to hourly and per-minute rates
- **Indirect costs** - Organisational overheads (rent, training, utilities, tools)
- **Journey mapping** - Time allocation across customer journey stages
- **Total cost analysis** - Cost per transaction and cost per customer
